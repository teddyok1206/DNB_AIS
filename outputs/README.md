# outputs

Runtime outputs live here but are ignored by git.

Track only small, curated summaries by copying them into `docs/`, `configs/`, or a dedicated experiment report. Keep checkpoints, GeoTIFFs, NetCDF files, DB files, tensor caches, and generated images out of git.

Density-model runs should use `outputs/dnb_density/` by default. See `docs/DNB_DENSITY_OUTPUT_WORKSPACE_20260602.md`.
