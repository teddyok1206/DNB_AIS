# 0409_0129

## 종료 전 기록

다음 진행 목표를 `sum-preserving edge-decay GT`로 확정했다.

배경:

- 현재 메인 supervision은 `y_edge_decay`
- 하지만 현재 방식은 퍼진 GT의 총합이 원본 선박 척수 합과 일치하지 않음
- 그래서 출력값을 그대로 `'척'` 단위로 해석하기 어렵다

다음 단계의 핵심은 다음과 같다.

- edge-decay spreading을 유지하되
- 선박 1척이 퍼진 뒤의 총합이 항상 `1`이 되도록 정규화
- 결과적으로 patch/scene 전체 합이 원본 GT 척수와 다시 연결되게 만들기

이번 기록은 다음 연구 일자에 바로 이어받기 위한 종료 메모다.
