# 0409_1129

## MPS 재테스트

- `DNB_AIS` 환경에서 MPS를 다시 점검했다.
- 결과:
  - `torch 2.9.1`
  - `torch.backends.mps.is_built() = True`
  - `torch.backends.mps.is_available() = False`
- 최소 tensor 생성 테스트도 실패했다.

실행 확인:

- `conda run -n DNB_AIS python -c "import sys, torch; ..."`
- `conda run -n DNB_AIS python -c "import torch; d=torch.device('mps'); x=torch.arange(..., device=d)"`

오류 요약:

- `RuntimeError: The MPS backend is supported on MacOS 14.0+. Current OS version can be queried using sw_vers`

추가 확인:

- `sw_vers` 결과는 `macOS 15.7.3`
- `platform.mac_ver()` 결과도 `15.7.3`, `arm64`

해석:

- OS 버전 자체는 조건을 만족한다.
- 그런데 현재 Codex 실행 환경에서는 MPS가 여전히 비활성으로 보인다.
- 즉 이번 세션 기준으로는 `torch`가 MPS로 실제 tensor를 올리지 못한다.

이번 턴은 상태 확인만 수행했고, 코드 변경은 없다.
