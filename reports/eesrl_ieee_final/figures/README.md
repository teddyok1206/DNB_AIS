# Figure Workspace

This directory keeps curated report figures separate from bulk runtime outputs.

## Folders

- `planned`: figures planned or under review (`넣을 예정`).
- `included`: figures currently intended for the report (`들어가 있음`).
- `removed`: figures tried but excluded (`넣었다가 뺌`).

Do not bulk-copy output directories here. Only copy or generate figures that are plausible report assets.
