# EESRL IEEE Final Report Workspace

This folder collects the active final-report materials.

## Structure

- `documents/`: report draft, prep notes, citation register.
- `templates/`: IEEE `.docx` template and Markdown drafting scaffold.
- `figures/`: curated figures, separated into planned/included/removed states.

## Source of Truth

Use `documents/FINAL_REPORT_DRAFT_IEEE.md` for drafting content. Transfer final text into `templates/Full-Paper-template.docx.docx` when ready.

Keep `org-*` template files at the repository root untouched as downloaded originals.
